
To compile, make sure jogl libraries are set up: javac *.java

To run: java ViewMesh

In-program commands:
'l': draw the lines, or edges, of the current mesh.
's': draw a shaded version of the current mesh.

'0': draw the current shape with zero subdivisions.
'1': draw the current shape with one subdivision.
'2': draw the current shape with two subdivisions.
'3': draw the current shape with three subdivisions.
'4': draw the current shape with four subdivisions.
'5': draw the current shape with five subdivisions.

'z': zoom in.
'x': zoom out.

't': draw a tetrahedron.
'q': draw a square.
'p': draw a pyramid.
